# MeloRhythm- Web based Music Player Application
Melorhythm is a web-based music player built with HTML, CSS, and JavaScript. It allows users to play, pause, and navigate songs through a responsive and user-friendly interface. The project demonstrates modern web development techniques and audio playback functionality in a browser.

## Project Overview
MeloRhythm is a modern web-based music player developed using HTML, CSS, JavaScript, and Flask. The application provides an interactive and user-friendly music listening experience with features such as playlists, favorites, recently played songs, mini player, fullscreen player, dark mode, and music visualization.

This project was developed as a Mini Project to demonstrate practical implementation of web development technologies and user interface design concepts.

## Objective
- To develop a web-based music player with a modern user interface.
- To provide seamless music playback and management features.
- To implement playlists, favorites, and recently played songs.
- To enhance user experience through responsive design and visual effects.
